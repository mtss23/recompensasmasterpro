Notificações

Receba lembretes para completar suas missões diárias

Horários de Notificação

Defina os horários em que deseja receber notificações

Nota importante: As notificações programadas funcionarão apenas quando o aplicativo estiver aberto no navegador. Para lembretes regulares, recomendamos usar os alarmes do seu dispositivo ou um aplicativo dedicado de lembretes.

Como Receber Notificações

1. No computador (Chrome)

Para receber notificações no Chrome:

  • Clique no ícone de cadeado na barra de endereço
  • Selecione "Notificações"
  • Escolha "Permitir"

2. No computador (Firefox)

Para receber notificações no Firefox:

  • Clique no ícone de informações na barra de endereço
  • Selecione "Permissões"
  • Ative "Enviar Notificações"

3. No celular (Android)

Para receber notificações no celular, adicione o Recompensas Master à tela inicial:

  • No Chrome, toque no menu (três pontos)
  • Selecione "Adicionar à tela inicial"
  • Confirme a adição

4. No celular (iOS)

Para receber notificações no iPhone:

  • No Safari, toque no ícone de compartilhamento
  • Selecione "Adicionar à Tela de Início"
  • Confirme a adição
